# AWS Database Table Operations Lab
## Overview
This project demonstrates hands-on experience with relational databases on AWS. It showcases the ability to create and manage databases and tables, perform schema modifications, and clean up resources using SQL commands in an AWS-managed environment.
Through this lab, I practiced essential database operations, including creation, modification, querying, and deletion of databases and tables, using the MySQL client on an EC2 instance.

## Objectives
By completing this lab, I demonstrated the ability to:
- Connect to an AWS EC2 instance with a database client 
- Create and drop databases and tables 
- View existing databases and table schemas 
- Modify table structure using SQL (ALTER) 
- Verify operations using SQL commands (SHOW)

## Lab Environment
- AWS Services Used: EC2, RDS (MySQL), MySQL client
- Database: MySQL 
- Instance: EC2 instance configured as a Command Host 
- Sample Data: Statistics Finland, regional statistics
