# Querying a Relational Database (MySQL)
## Overview
Completed a hands-on lab in AWS focused on querying a relational database using SQL. The lab involved interacting with a pre-configured MySQL database (world) hosted on an AWS EC2 Command Host instance.
The objective was to practice data retrieval, filtering, and analysis using core SQL operations in a cloud environment.

## Technologies & Tools
- AWS EC2 (Command Host via Session Manager)
- MySQL
- SQL (Data Querying Language)

## Dataset
The database world included three tables:
- country
- city
- countrylanguage <br>
Data source: Statistical datasets (regional/global metrics such as population, surface area, etc.)
